# 🏗️ Stack Up!

Viral bir kule yığma oyunu. Blokları üst üste yığ, en yüksek kuleyi inşa et!

## 🎮 Nasıl Oynanır?

- **Tıkla / Dokun / Space** → Bloğu bırak
- Blok tam hizalıysa → **PERFECT!** 🔥
- Art arda perfect → **Combo** → Blok büyür!
- Taşan kısım düşer → Platform küçülür
- Ne kadar yüksek, o kadar hızlı!

## ✨ Özellikler

- 🎯 Tek dokunuşla oynanan basit mekanik
- ✂️ Kesme mekaniği — taşan kısım düşer
- ✨ Perfect + Combo sistemi — blok büyür
- 🌈 Rainbow renk paleti
- 🌟 Parallax yıldızlar + bulutlar
- 💥 Partikül efektler
- 🏆 Yüksek skor (localStorage)
- 📤 Skor paylaşım (Web Share API)
- 📱 Mobil uyumlu

## 🛠️ Teknoloji

- **Phaser 3** — 2D oyun motoru
- **TypeScript** — Tip güvenli geliştirme
- **Vite** — Hızlı build
- **Howler.js** — Ses efektleri
- **@tweenjs/tween.js** — Animasyonlar

## 🚀 Çalıştır

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
